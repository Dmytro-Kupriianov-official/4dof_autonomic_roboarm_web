# 4dof_autonomic_roboarm_web
ESP32 robotic arm (3→4 DOF) with onboard kinematics, supporting autonomous playback of recorded motions and remote control via web interface or PC. Modular design ready for sensor integration and logic-based behavior.

ESP32-based robotic arm (3 DOF, planned 4 DOF) with onboard kinematics, autonomous playback of recorded motions, and remote control via web/PC. Planned: sensor integration, logic-based behavior, improved UI, and full standalone operation.

Current prototype includes servo control (PWM driver), DC-DC converter, battery power, potentiometer input, buzzer, and small display. Built as a cardboard proof-of-concept. Further development will continue after fixing the main PC and completing the 3D printer.
